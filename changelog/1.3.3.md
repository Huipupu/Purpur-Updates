# Purpur 1.3.3

- 新增后端自动休眠与自动待机关闭，减少空闲时显存和系统资源占用。
- 优化后端启动与环境状态检测，减少重复安装、重复修复和配置状态异常。
- 优化 GPU / CPU 环境切换与配置记录，提升重启后的设备状态一致性。
- 新增字幕整理优化功能，支持按目标字数智能优化字幕拆条结果。
- 新增整理优化提示词自定义，用户可按需求调整字幕整理规则。
- 优化字幕校对流式输出，生成过程反馈更及时。
- 优化字幕对齐与导入流程，提升字幕文件命名和重复对齐稳定性。
- 新增字幕历史音频再调用/复用，已提取音频可直接恢复继续处理。
- 新增历史字幕恢复到检查字幕界面，可同步调用对应音频和字幕内容。
- 新增 Purpur 标题任务加载动画，执行任务时显示粒子动效。
- 优化启动、唤醒和任务等待提示，减少用户等待过程中的困惑。
- 更新插件信息说明，补充字幕校对、整理、对齐与导入功能介绍。
